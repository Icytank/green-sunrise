# Green Sunrise

Professional PV Installation & Machinery Specialists based in Bulgaria, offering utility-scale PV installation services across Europe.

**Website**: [https://green-sunrise.net](https://green-sunrise.net)

## Technology Stack

This website is built with modern, high-performance web technologies:

- **Framework**: [Astro](https://astro.build/) (v5) for static site generation and zero-JS default performance.
- **CMS**: [Decap CMS](https://decapcms.org/) (formerly Netlify CMS) for easy content management directly via Git.
- **Styling**: LESS / CSS for custom, lightweight design.
- **Hosting**: Hosted for free on **Cloudflare Pages**, ensuring fast global delivery and automatic deployments.
